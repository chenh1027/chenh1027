HTML Tags I Learned from the Web Course

These Are Some Tags I Think Are Easy to Confuse

ol
"ol" is for making a list with an order. For example: when we have a competition in the rankings.
ul
"ul" is the opposite of "ol", which is for making an unordered list. For example: making a name book of our class.
li
"li" is used for items in the list. We can have multiple "li" elements for one list.
dl
"dl" is also a list, but unlike "ol" and "ul", it can have descriptions under the list.
dt
"dt" is associated with "dl", and is used to define terms listed in "dl".
